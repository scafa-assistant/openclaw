# App Store Screenshots Guide

## Required Screenshots

### iPhone (Required: 6.5" and 5.5" or 6.7" and 5.5")
1. **Home Screen**: Main chat interface
2. **Voice Interaction**: Voice command in action
3. **Settings**: AI model selection, preferences
4. **History**: Chat history view
5. **Siri Shortcut**: Siri integration demo

### iPad (Required if supporting iPad)
1. **Home Screen**: iPad-optimized chat view
2. **Split View**: Multitasking support
3. **Settings**: iPad settings panel

### Android (Required: Phone and 7" + 10" tablets)
1. **Home Screen**: Material Design chat interface
2. **Voice Interaction**: Voice search demo
3. **Settings**: Preferences and model selection
4. **Widget**: Home screen widget preview
5. **Shortcuts**: App shortcuts showcase

## Screenshot Specifications

### iOS
- **6.5" Display**: 1284 x 2778 pixels (iPhone 14 Pro Max)
- **5.5" Display**: 1242 x 2208 pixels (iPhone 8 Plus)
- **iPad Pro**: 2048 x 2732 pixels
- Format: PNG or JPEG
- No transparency

### Android
- **Phone**: 1080 x 1920 or 1080 x 2400
- **7" Tablet**: 1080 x 1920
- **10" Tablet**: 1440 x 2560
- Format: PNG or JPEG
- Max 8 MB per screenshot

## Screenshot Content Guidelines

### Do:
- Show real app content (not mockups)
- Use German and English text
- Demonstrate key features
- Keep it clean and focused
- Show diversity in content

### Don't:
- Include device frames (unless required)
- Add marketing text overlays
- Show status bars with personal info
- Include pricing in screenshots
- Use placeholder content

## Feature Graphic (Android)
- Size: 1024 x 500 pixels
- Safe zone: Text within 924 x 400 (center)
- Format: PNG or JPEG
- No transparency

## App Icon
- **iOS**: 1024 x 1024 (App Store), multiple sizes for app
- **Android**: 512 x 512 (Play Store), adaptive icons for app
- Format: PNG
- No transparency (except Android adaptive)
